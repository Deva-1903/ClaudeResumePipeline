# Interview Prep Skill

## Goal

Prepare Deva for an interview using the exact tailored resume he applied with.

## Rules

- Use only when explicitly requested.
- Read the specified Tailored_Resume.tex.
- If a JD is provided, use it.
- If no JD is provided, prepare from the resume alone.
- Do not modify the resume unless explicitly asked.
- Do not generate PDF.
- Do not create files unless explicitly asked.
- Keep claims truthful.

## Output

Produce:
- likely interview themes
- project talking points
- resume bullet defense notes
- technical questions to prepare
- behavioral stories to prepare
- weak spots or risky claims

Be direct and practical.
