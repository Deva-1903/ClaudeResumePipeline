# AI Software Engineer - Intern

**Company:** Aira
**Location:** Saratoga, CA
**Employment Type:** Intern

## About Aira

We build AI agents that run cellular networks.

Mobile operators spent billions on 5G but can't extract value — vendor lock-in blocks innovation, monolithic systems don't scale, operational complexity multiplies. We deploy multi-agent systems with fine-tuned LLMs that autonomously optimize network performance, generate operational tools, and execute operator intent. Applied AI at planetary scale: millions of towers, petabytes of telemetry, real-time inference affecting billions of users.

Founded by Qualcomm and Intel veterans. Team from Stanford, Berkeley, Cornell. Backed by AT&T Ventures, Snowflake, Intel Capital, In-Q-Tel, NeoTribe, Acrew, and more.

## The Role

Build the product layer for our agent platform — interfaces where network operators interact with AI systems, UIs surfacing agent decisions, and the APIs connecting LLMs to real-time network data. Full-stack, fast-moving, and directly collaborative with ML researchers, wireless engineers, and designers.

## What You'll Build

- Interactive UIs visualizing network topology, coverage, and live agent decisions
- APIs connecting fine-tuned LLMs and multi-agent systems to frontend experiences
- Real-time dashboards rendering high-volume telemetry and agent-generated insights
- Orchestration components — tool integrations, LLM workflows that bridge models and product surfaces
- Infrastructure integrations across cloud services, containers, CI/CD, and observability systems

## What We're Looking For

- Pursuing a Bachelor's, Master's, or PhD in CS, Software Engineering, or related field
- Strong software engineering fundamentals: data structures, APIs, databases, debugging, testing, and system design basics
- Strong across modern web stacks: Python, React (familiarity with Maps integration), Node.js
- Experience building data-intensive UIs with complex state management
- Familiarity with LLMs, agent orchestration, and how AI outputs become product experiences; LangGraph or Autogen experience is a plus
- Familiarity with cloud infrastructure, Docker, CI/CD, and development workflows
- Pragmatic, collaborative, and product-minded: able to move fast, ask good questions, and ship working software
